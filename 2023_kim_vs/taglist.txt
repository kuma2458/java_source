form : 로그인칸 같이 무언갈 입력받거나 출력하는 틀
    ㄴmethod : 데이터를 어떻게 전송할지 정하는 태그
        ㄴpost : http나 https의 body에 포함되어 데이터 전송
        ㄴget : url이랑 입력하거나 출력한 데이터를 합쳐서 한번에 보냄
            이때는 url에 데이터가 보여짐
input : 입력받는 칸
    ㄴname : input의 이름
    ㄴtype : 입력받는 데이터 종류, text, password, submit 등 있다
    ㄴplaceholder : 입력 전 미리표시되는 텍스트를 넣는 태그 
    ㄴrequired : 반드시 입력하게 하는 태그
    ㄴreadonly : 읽기만되는것, 수정x, 다만 서버로 전송은됨
    ㄴdisabled : 수정x, 서버로 전송x
    ㄴautofocus : 새로고침시 자동으로 커서가 가는거
    ㄴlabel : 텍스트에 라벨을 붙이듯 누르면 for이 가리키는걸 선택해줌
    ㄴchecked : 기본적으로 선택되게 해주는 값
    ㄴminlength :최소입력길이
    ㄴmaxlength :최대입력길이
    ㄴlist : datalist 가리키는 태그
